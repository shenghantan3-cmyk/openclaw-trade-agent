# 插件注册配置与测试用例执行记录

## 一、插件注册配置说明
### 1. 插件目录结构
```
~/.openclaw/plugins/
└── trade-order/
    ├── __init__.py
    ├── generate_trade_order.py  # 核心插件代码
    └── trade_agent.py          # Agent逻辑
```

### 2. OpenClaw配置文件修改
编辑 `~/.openclaw/config.yaml`，添加以下配置：
```yaml
plugins:
  trade-order:
    path: ~/.openclaw/plugins/trade-order
    entry: generate_trade_order.generate_trade_order
    description: 量化交易订单生成插件
    version: 1.0.0
    author: OpenClaw
```

### 3. 注册验证
执行命令查看已注册插件：
```bash
openclaw plugins list
```

**预期输出：**
```
┌───────────────┬─────────────────────────────────┬─────────┐
│ Name          │ Description                     │ Version │
├───────────────┼─────────────────────────────────┼─────────┤
│ trade-order   │ 量化交易订单生成插件            │ 1.0.0   │
└───────────────┴─────────────────────────────────┴─────────┘
```

**测试输入：**
